Минималистичный кастомный MOTD-скрипт для Linux с использованием 256-цветного ANSI-режима и градиентов.  

## Описание

Скрипт выводит информативный MOTD при логине в shell:
- FQDN хоста (градиент)
- ОС
- Версию ядра
- Основной IP-адрес
- Аптайм
- Load Average
- Использование памяти
- Использование корневого раздела

Визуально оформлен с помощью цветных градиентных линий и разделителей.

## Особенности

- Использует `tput setaf` (256 цветов)
- Градиенты для:
  - верхней и нижней рамки
  - разделителей
  - FQDN хоста
- Совместим с Debian и RHEL-подобными системами
- Без сторонних зависимостей

## Требования

- Терминал с поддержкой `xterm-256color`
- Bash
- Утилиты: `hostname`, `uptime`, `free`, `df`, `tput`

## Установка

```bash
cp motd.sh /etc/profile.d/motd.sh
chmod +x /etc/profile.d/motd.sh
```

## Если появиться желание отредактировать под себя  
Выполни команду, которая даст номера цветов в палитре.  

```bash
for i in {0..255}; do printf "\033[38;5;%sm%3s " "$i" "$i"; done; echo
```
